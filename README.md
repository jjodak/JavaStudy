### 📝 Repository Overview
이 저장소는 자바 프로그래밍의 기초부터 심화 객체지향 개념, 자료구조, 예외 처리까지 단계별로 학습한 코드를 기록하는 공간입니다. **`신용권의 이것이 자바다`** 교재와 학과 커리큘럼을 중심으로 실습을 진행하였습니다.

| 주차 | 주요 학습 내용 | 핵심 키워드 |

| **W4** | 조건문과 반복문 | if-else, switch, for, while, break/continue |

### 🚀 주요 프로젝트 & 실습 (Highlights)
* **FightSimulator:** 상속과 인터페이스를 활용한 턴제 스킬 배틀 게임 엔진 구현.
* **Library Management:** `ArrayList`를 활용한 도서 정보 관리(추가, 삭제, 검색) 시스템.
* **Calculator Test:** 인터페이스와 익명 객체를 활용하여 유연한 사칙연산 로직 구현.
* **Bank System:** 예외 처리를 적용하여 잔고 부족 시 예외를 발생시키는 출금 시스템.

### 📂 폴더 구조 (Project Structure)
* `Javaw2` ~ `Javaw14`: 주차별 실습 코드 및 예제.
* `ArrayList test2`: 컬렉션 프레임워크 심화 실습.
* `FightSimulator`: 객체지향 원리를 적용한 종합 게임 프로젝트.
